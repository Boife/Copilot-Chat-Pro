## 安裝

1.  前往[發布頁面](https://github.com/qzxtu/Copilot-Chat-Pro/releases)並下載最新版本。
2.  解壓縮下載的 zip 檔案。
3.  打開瀏覽器並導航至：
    -   **鉻合金：**`chrome://extensions/`
    -   **邊緣：**`edge://extensions/`
4.  啟用右上角的「開發者模式」。
5.  點擊「載入已解壓縮的檔案」並選擇步驟 2 中解壓縮的資料夾。

## 用法

1.  開啟 Bing 或 Copilot 網站。
2.  聊天時無需擔心字元限制。
3.  好好享受！
