## 安装

1.  前往[发布页面](https://github.com/qzxtu/Copilot-Chat-Pro/releases)并下载最新版本。
2.  解压下载的 zip 文件。
3.  打开浏览器并导航至：
    -   **铬合金：**`chrome://extensions/`
    -   **边缘：**`edge://extensions/`
4.  启用右上角的“开发者模式”。
5.  单击“加载已解压的文件”并选择步骤 2 中解压的文件夹。

## 用法

1.  打开 Bing 或 Copilot 网站。
2.  聊天时无需担心字符限制。
3.  好好享受！
